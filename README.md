# Appetizer
Appetizer is a project to practice and learn iOS development. Learning how to fetch JSON data, caching and showing alerts,
is the main goal of this App.

## Used Technologies
- Swift 5.5
- SwiftUI
- Codable
- JSON
- Combine (for getting JSON data)

### SwiftUI
- Form, Section
- ProgressView
- TabView
- Toggle
